# LinPriX
# Linux Privilege Escalation Analyzer

A modular Python-based tool to detect common Linux privilege escalation vectors.

## Features

- Sudo misconfiguration detection
- SUID binary enumeration
- Group-based privilege analysis
- Cron job inspection
- Writable file detection
- PATH misconfiguration checks
- Final summary of attack vectors

## Usage

```bash
python3 main.py
